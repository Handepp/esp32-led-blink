# ESP32 LED Blink

Basic ESP32 project demonstrating non-blocking LED blinking using `millis()` instead of `delay()`.

---

## 🚀 Features

- Blink onboard LED using GPIO  
- Non-blocking timing with `millis()`  
- Serial monitor logging  
- Beginner-friendly ESP32 example  

---

## 🔌 Hardware

- ESP32 Dev Board  
- Onboard LED (GPIO 2)

---

## ▶️ How to Use

1. Open `main.ino` in Arduino IDE  
2. Select your ESP32 board  
3. Upload the code  
4. Open Serial Monitor (115200 baud)

Example output:
=== ESP32 LED Blink System Started ===
[STATE] LED ON
[STATE] LED OFF
